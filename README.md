# orng Luuzu

orng Dark theme for [Zed](https://zed.dev), Luuzu fork of the orng theme.

## Install (dev extension)

1. Open the Extensions page in Zed.
2. Click **Install Dev Extension**.
3. Select this directory.

Then pick **orng Dark -- Luuzu** in the theme selector.
